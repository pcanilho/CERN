NQView

NQView is the middle-ware software which is used to:

 

 

Main Usage

  • Control the remote monitoring probes;
  • Troubleshoot the system behavior in case of mobile network failure, subscribers, tasks & jobs to be ran in the probes;
  • FTP/SMS server configuration setup.
  • Create probes in the system;
  • Attach a subscriber, application level alarms;
  • Assign task/job/campaign.

NQDI

NQDI acts as a deep analysis or troubleshooting of past events. The troubleshooting steps can vary, from a simple calculation to a monthly statistics report , e.g.:

  • Call setup time calculation using the captured layer 3 messages during a job;
  • Detect sudden changes in network signal strength levels during a time of the day, or zone in the case of a drive test;
  • End user network performance assessment.

NQWeb - System Inspector

System Inspector module of NQWeb is responsible for the dashboard and general network overview system. With this module the user can achieve the following:

Dashboard - SQL Query in HTML Elements

TABLE

MAP

GRAPH

CHART

Software Package - NQWeb - System Inspector

  • Display the result of a SQL query in a table HTML element;
  • Add performance overview graphs and charts of a specific protocol or network feature, e.g. FTP, Dropped Calls, etc;
  • Analyse the overall network performance with a time feature.

Disadvantages

    • No technology filter;
    • Dashboards cannot be deleted if a mistake is made during their creation;
    • No custom design/styling is possible on the HTML elements.
